この画面は初めての起動時、アップデート後、設定初期化後の起動時に表示されます。閉じても問題ありません。

OLCVの使い方

立ち上げると通知領域（デフォルトでは右下のスピーカーマークがある枠）にこのアプリのアイコンが表示されます。

クリックするとメニューが表示され、設定からURLやCSS、コメント欄の位置と大きさを変更できます。

URLの欄にはYouTubeLiveチャット欄のURLを貼り付ける必要があります。視聴画面からはチャット欄右上の3つの点が並んだボタンをクリックし、「チャットをポップアップ」を押し表示されたウインドウのURL欄のURLを貼り付けて下さい

最新の情報はREADME.mdを参照してください。

現在のバージョンには自動更新機能はないため手動でのアップデートが必要です。（実装予定。優先度高めに作業します）

既に報告されていないバグや機能要望などなにかあれば@Len_Takayamaに連絡かIssueを開いてください。
